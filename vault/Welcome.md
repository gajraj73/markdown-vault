# Welcome to Markdown Vault

Your personal Markdown file manager.

## Getting Started

- **Create files** using the + button in the top bar or sidebar
- **Upload files** by clicking the upload button or dragging .md files into the app
- **Organize** with folders, tags, and favorites
- **Search** across all files with Cmd+P

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+P` | Open search |
| `Cmd+S` | Save current file |

## Features

- Split editor + preview
- Syntax highlighting for code blocks
- GFM support (tables, task lists, strikethrough)
- Dark/light mode
- Tags and favorites
- Full-text search
